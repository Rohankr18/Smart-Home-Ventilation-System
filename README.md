# Smart-Home-Ventilation-System
Smart Home Ventilation System Using ESP32

In this project, a Smart Home Ventilation System has been designed and developed with the primary goal of maintaining optimal indoor air quality by intelligently managing humidity levels. At the core of the system lies the ESP32 microcontroller, a powerful and energy-efficient platform that enables both real-time processing and wireless communication. The system employs a DHT11 sensor to continuously monitor the ambient temperature and humidity within an indoor environment. Based on the sensed humidity data and a predefined threshold value, the ESP32 dynamically controls an SG90 servo motor, which mechanically opens or closes a ventilation flap to regulate air exchange within the room.
